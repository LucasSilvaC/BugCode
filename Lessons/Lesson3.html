Alertas e avisos

Alertas são recursos muito utilizados em JS porque com ele é possível criar caixas de diálogos que aparecem na tela do usuário.

Como fazer um alerta?

Utilizando uma única linha de código é possível criar um alerta, veja e tente você mesmo. Digite o que quiser.

Script.js

alert("Olá, mundo!");

Experiência do usuário

A função de alerta para o código inteiro quando é requisitada, além de ter um design básico, preferencialmente é recomendado para necessidades críticas, pelo seu grau de importância, por mais que possa vir a ser usado de qualquer forma.

Alternativas ao uso de alertas

Atualmente existem diversas alternativas para os alertas, até pensando em dispositivos móveis, utilizar caixas controladas em CSS pode ser uma boa alternativa. Mas não pode ser descartado o uso do Alerta, que é simples e ótimo para mensagens importantes.
