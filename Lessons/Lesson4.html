O que são variáveis

Variáveis são espaços em branco usados para armazenar dados. Elas permitem você a você criar dados e armazenar na memória do computador. Elas podem ser acessadas ao longo do código se necessário.

Tipos de variáveis

Existem três formas de você criar variáveis em Javascript, utilizando o var o let e const , clicando em cada uma delas é possível ver suas especificações. Compreender esses conceitos é fundamental para a construção de aplicações JavaScript robustas e eficientes.

Tipos de dados

Em JavaScript, as variáveis podem armazenar diferentes tipos de dados, como números, strings (texto), booleanos (verdadeiro ou falso), arrays (listas) e objetos (coleções de pares chave-valor). Vale lembrar que o Javscript é uma linguagem tipada .

Script.js

// Booleano
var ativado = true;

// Int (Inteiro)
let altura = 180;

// Float (Número quebrado)
var taxa = 99.9;

// String (Texto)
var nome = "Isaque";

// Array (Lista)
let frutas = ["Banana","Laranja"];

Boas práticas ao trabalhar com variáveis

Nomear suas variáveis de forma clara e descritiva é uma boa prática, pois facilita a leitura e a manutenção do código. Por exemplo, ao invés de usar um nome genérico como x, é melhor usar idade ou nomeUsuario. Além disso, ao declarar variáveis, é importante escolher entre let e const dependendo se você planeja mudar o valor da variável mais tarde. Usar const quando possível ajuda a evitar erros e torna o código mais previsível.
