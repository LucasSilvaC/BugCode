O Que São Condicionais?

Condicionais são estruturas de controle que permitem que um programa tome decisões com base em condições. Em JavaScript, as principais estruturas condicionais são if, else if e else, que executam um código específico apenas se uma condição for verdadeira. Isso torna o programa dinâmico, reagindo de acordo com a entrada ou estado atual.

IF

A estrutura if em JavaScript permite que o código execute uma ação apenas se uma condição for verdadeira. Se a condição dentro dos parênteses (()) for verdadeira, o bloco de código dentro das chaves ({}) será executado.

Script.js

let idade = 18;
if (idade >= 18) {
alert("Você é maior de idade.");
}

Else If

Quando você precisa verificar várias condições, pode adicionar quantos blocos else if forem necessários. Cada else if verifica uma condição após o if inicial e só será executado se todas as condições anteriores forem falsas.

Script.js

let idade = 18;
if (idade > 18) {
alert("Você é maior de idade.");}
else if (idade == 18){
alert("Você acabou de se tornar maior de idade.");}

Else

O else é usado para capturar qualquer condição que não tenha sido atendida pelos if e else if anteriores. Ele é executado apenas se todas as condições anteriores forem falsas. Em casos de múltiplas condições, o else garante que uma ação será tomada caso nenhuma outra condição seja atendida.

Script.js

let idade = 18;
if (idade > 18) {
alert("Você é maior de idade.");}
else{
alert("Você não é maior de idade.");}
