Importância de um código limpo

O famoso "cleancode" é essencial para um bom código JavaScript, com isso, é sempre importante manter uma boa organização com seus códigos.

Como fazer um cleancode?

Cleancode é uma boa prática da programação de priorizar a estrutura e o bom uso do código, deixando o de forma limpa e compreensiva, definindo nomes realista de variáveis e sempre comentando o código.

Comentários

Uma forma fácil de organizar o código é comentando ele, desde a sintaxe a funções específicas, comentários não alteram nada no código e são "invisiveis" ao executar, existem vários tipos de comentários.

Tipos de comentários

No JavaScript existem dois tipos de comentários, de linhas únicas, e os que são ilimitados. Eles respectivamente são "//" e o "/* */".

Script.js

/* Aula 02 de JS */

// Aula 02 de JS

/*
Aula 02 de JS
