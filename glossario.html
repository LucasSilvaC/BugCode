Glossário

Palavra Tipo Significado Link
Alertas Comando em Javascript São caixas de diálogo visíveis ao usuário para exibir algo. Link
Array Estrutura de Dados Armazena um conjunto de dados que podem ser acessados por índices. Link
Banco de dados Armazenamento Conjunto de dados organizados e armazenados para fácil acesso e manipulação. Link
Booleano Tipo de Dado Tipo de dado que representa dois valores possíveis: verdadeiro (true) ou falso (false). Link
Char Tipo de Dado Tipo de dado que armazena um único caractere. Link
CleanCode Prática de Programação Conjunto de princípios e boas práticas para escrever código limpo, legível e fácil de manter. Link
Comentários Documentação Notas no código que são ignoradas pelo compilador, usadas para explicar ou documentar partes do código. Link
Condicional Estrutura de Controle Estruturas que permitem a execução de diferentes blocos de código dependendo de uma condição. Link
Const Variável Uma variável cujo valor não pode ser alterado após sua declaração. Link
CSS Folha de Estilos É usado para estilizar e definir o layout de páginas web, controlando o formato e o estilo de elementos HTML. Link
ELSE Estrutura Condicional Parte de uma estrutura condicional que define o que fazer caso a condição inicial seja falsa. Link
Front-End Desenvolvimento Web Parte do desenvolvimento web que lida com a interface do usuário, como o design e a interação com o usuário. Link
HTML Estrutura de Páginas Web Linguagem de marcação usada para criar e estruturar o conteúdo das páginas na web. Link
IF Estrutura Condicional Usado para realizar uma verificação de condição e executar um bloco de código se a condição for verdadeira. Link
Incremento Operação Aritmética Operação que aumenta o valor de uma variável, geralmente em 1. Link
JavaScript Linguagem de Programação Linguagem de programação usada para criar interatividade nas páginas web e para controlar o comportamento do site. Link
Let Variável Uma variável cujo valor pode ser alterado após a declaração, com escopo de bloco. Link
String Tipo de Dado Tipo de dado que armazena uma sequência de caracteres (texto). Link
Var Variável Uma variável cujo valor pode ser alterado após a declaração, com escopo global ou de função. Link
Variáveis Elemento de Armazenamento Espaços de memória usados para armazenar dados temporários durante a execução do programa. Link
